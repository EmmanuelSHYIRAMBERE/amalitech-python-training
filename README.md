##  Concepts Covered

### Language Fundamentals
- Core data types: `str`, `int`, `float`, `bool`, `list`, `dict`, `set`, `tuple`
- Control flow: `if/elif/else`, `for`, `while`
- List/dict/set **comprehensions**
- Exception handling: `try/except/raise`
- Type hints (`str | None`, `list[Student]`)

### Functions & Modules
- Reusable functions with clear single responsibilities
- `*args` and `**kwargs` for flexible signatures
- Lambda expressions as sort keys
- Module organization and `__init__.py` package exports

### Object-Oriented Programming
- **Abstract Base Classes** (`ABC`, `@abstractmethod`) to enforce interfaces
- **Inheritance** — base class + specialized subclasses (`super()`)
- **Encapsulation** — `@property`, `@setter`, name-mangled attributes (`_private`)
- **Polymorphism** — same method call, different behavior per subclass
- **Dunder methods** — `__str__`, `__repr__`, `__eq__`, `__hash__`
- `@classmethod` factory methods and `@staticmethod` utilities

### Tooling & Environment
- `venv` / `poetry` for virtual environments and dependency management
- `requirements.txt` / `pyproject.toml`
- Git version control and GitHub workflow
- VS Code / PyCharm debugging basics
